# Health-insurance-analysis-dashboard
Power BI dashboard analyzing insurance cost, lifestyle factors, and demographic insights

## 📌 Project Overview

This project focuses on analyzing **health insurance data** to uncover how **lifestyle, demographic, and regional factors** influence insurance costs.

The goal is to transform raw data into **meaningful insights** using interactive visualizations.

---

## 🎯 Objectives

* Analyze insurance cost patterns
* Understand impact of smoking, alcohol & exercise
* Study cholesterol levels and health indicators
* Explore gender and age-based trends
* Identify location-wise variations

---

## 📊 Dashboard Highlights

### 🔹 Executive Overview

* Total insurance cost
* Average cost insights
* Key KPIs for quick understanding

### 🔹 Behavior & Risk Analysis

* Smoking vs cost analysis
* Exercise impact on insurance
* Cholesterol-level insights

### 🔹 Demographic & Location Insights

* Gender-based comparison
* Age group analysis
* Location-wise cost distribution
